Here the events are 3 types 1. Browser Events 2. Mouse Events 3. Form Events Browser Events - onload() Mouse Events 1. onfocus() - this will trigger when you land on the particular input field 2. onchange() - this will trigger when you enter input and leaves the input field 3. oninput() - this will trigger when you enter any value within a field 4. onblur() - this will trigger when you visit the input field and leaves
